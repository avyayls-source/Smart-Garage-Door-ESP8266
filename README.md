# Smart-Garage-Door-ESP8266
IoT based Smart Garage Door Control System using ESP8266 with WiFi.  Includes circuit design, PCB Gerber files, simulation, and source code.
# Smart Garage Door Control System using ESP8266

## Aim
To design and develop a smart garage door control system using ESP8266 with WiFi capabilities.

## Problem Statement
Traditional garage doors require manual operation and lack remote monitoring and automation features.

## Scope of the Solution
The system allows remote monitoring and control of a garage door using ESP8266 and WiFi connectivity.

## Required Components
Hardware:
- NodeMCU ESP8266
- Relay Module
- Slide Switches
- LED
- Resistor
- Terminal Blocks
- Jumper Wires

Software:
- Arduino IDE
- EasyEDA
- Wokwi Simulator

## Simulated Circuit
Simulation performed using Wokwi.

## PCB Design
PCB designed using EasyEDA.

## Gerber Files
Gerber files for PCB manufacturing are included in this repository.

## Code
ESP8266 Arduino code for controlling the relay and switches.
